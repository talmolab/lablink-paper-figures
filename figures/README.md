# Figures

This directory contains generated figures for the LabLink paper.

## Structure

- `main/` - Figures for the main text
- `supplementary/` - Supplementary figures

## File Naming Convention

Use descriptive names that indicate the figure number and content:
- `figure_1_system_architecture.pdf`
- `figure_2_performance_comparison.pdf`
- `supplementary_figure_1_detailed_metrics.pdf`

## Format Guidelines

- Save publication-ready figures as PDF or SVG (vector formats)
- Save PNG versions for quick preview (300 DPI minimum)
- Include both formats when possible
